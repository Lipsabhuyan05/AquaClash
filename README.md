# AquaClash 

A simple Python command-line game based on Snake–Water–Gun logic.

## How to Play
1. Run the program
2. Enter number of rounds
3. Choose between:
   - snake
   - water
   - gun

## Rules
- Snake drinks Water
- Water damages Gun
- Gun kills Snake

## Tech Used
- Python
- random module
